# Быстрая настройка

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Создание .env файла

Создайте файл `.env` в корне проекта и заполните его:

```
BOT_TOKEN=your_bot_token_here
NOTIFICATION_CHANNEL_ID=your_channel_id_here
```

(образец в файле `env_example.txt`)

## 3. Получение токена бота

1. Откройте @BotFather в Telegram
2. Отправьте `/newbot`
3. Укажите имя и username бота
4. Скопируйте токен в `.env`

## 4. Получение ID канала

**Способ 1 (простой):**
1. Создайте канал
2. Добавьте бота @raw_data_bot в канал
3. Он пришлёт ID канала

**Способ 2 (через API):**
1. Добавьте вашего бота в канал как администратора
2. Отправьте сообщение в канал
3. Откройте: `https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates`
4. Найдите `"chat":{"id":-1001234567890,...}`
5. Скопируйте ID (с минусом!) в `.env`

## 5. Запуск

```bash
python bot.py
```

## Проверка

1. Найдите вашего бота в Telegram
2. Отправьте `/start`
3. Проверьте, что:
   - Бот ответил с тремя кнопками
   - В канал пришло уведомление о новой регистрации
